this page contains cart page

The user will be able to click on the basket (or any button like “show my basket”) to open a new page and see the detailed list of products and total price. There will be the possibility to Delete a product Empty the basket Pay (*) Continue shopping / return to the store (*) we are not implementing any kind of user authentication, payment etc… therefore this function should simply update the store (product quantities) and present the confirmation of payment with the total amount displayed. Then the user will be able to return to the store with an empty shopping basket. Note that by the time the user completes the payment, the stock might change. If a product is not available anymore, you may either cancel the payment with a relevant message (for example returning to the shopping basket and asking the user to delete the product first). Be careful to leave the database in its initial state! or proceed to payment while removing the missing products. Here also you may need to cancel the payment with a relevant message if the amount to pay is zero (for example if all products are out of stock).

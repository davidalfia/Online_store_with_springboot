The landing page (“/”) of the website will present the store and 5 products on sale (select the 5 top products with biggest discount). Each product will be presented with its name, picture and price after discount (price – price*discount/100) for example in the style of https://www.bookdepository.com. The images should be resized to match a fixed size for a consistent presentation (do not assume images are all the same size). There should be a search bar where the user can enter a (any substring or all of a) product/book name, and the page will display the search results. The user will be able to add products to his shopping basket. It is advised to add a button near each product. The shopping basket will appear as a small icon/button on the side with the current number of items in baskets. The user will be able to click on the basket (or any button like “show my basket”) to open a new page and see the detailed list of products and total price. There will be the possibility to Delete a product Empty the basket Pay (*) Continue shopping / return to the store
